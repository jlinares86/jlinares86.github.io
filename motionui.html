Motion UI

OVERVIEW

Motion UI is a Sass library for quickly creating CSS transitions and animations.
It also includes a bundle of pre-made effects as CSS classes.
This includes transition effects to slide, fade, hinge, scale, and spin, as well as a few built-in animations.

Pros

  • people are instinctively drawn to movement
  • it is easier to understand and encourages the user to keep searching, exploring and discovering
  • Animations can provide context and explain where the user is going

Cons

  • Not all browsers handle the code the same way
  • The animations are not free and add additional weight and data usage to the product
  • Designers need to be aware of technical limitations and collaborate with developers so the site performs efficiently

Examples:

You can apply these transitions to overlays, off-canvas menus, modals, and more.

All animations are created with the mui-animation() mixin.

Animation effects can be played in series.

The three main types of CSS classes of Motion UI are:

  1. Transition class
  2. Animation class
  3. Modifier class
